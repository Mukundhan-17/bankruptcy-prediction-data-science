Bankruptcy Prediction Dashboard (Machine Learning Application)

This repository contains a production-ready machine learning application designed to assess bankruptcy risk for companies using financial and operational indicators.
The solution integrates model training, probability-based risk scoring, explainable analytics, and an interactive decision-support dashboard built with Streamlit.

Project Objective

The goal of this project is to support credit risk and loan approval decisions by:

Predicting the likelihood of company bankruptcy

Quantifying risk using probability scores

Translating model outputs into clear, actionable loan recommendations

Presenting insights in a business-friendly and interpretable format

Key Capabilities

Binary classification model to predict:

Bankruptcy

Non-Bankruptcy

Probability-based risk scoring for informed decision-making

Loan recommendation logic aligned with banking use cases

Interactive dashboard for non-technical users

CSV-based batch predictions and manual input support

Explainable feature impact analysis

Dynamic summary metrics and insights
